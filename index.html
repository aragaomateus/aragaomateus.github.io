About Me

Computer And Data Science Student

As a recent Computer and Data Science graduate from NYU (December 2023), my internships have shaped my career aspirations. At Moises.ai, I discovered a passion for data visualization, enabling me to transform complex analyses into clear, actionable insights. This skill was further honed during my internship with the AMC Paid Features team. There, I built a data observability framework for Spark applications, designed analytics dashboards for runtime metrics, and implemented cloud infrastructure for data operations using AWS CDK.

Beyond technical skills, I prioritize effective communication. During my last 2 years in college, I've served as an Intro to Python Programming tutor at NYU, refining my ability to explain intricate concepts to beginners. This experience translates directly to data analysis roles, where clear communication is key to stakeholder understanding.

Excitedly, I'm starting a new position as a Junior Lab Associate at NYU's Liao Lab, beginning soon. In this role, I'll support Professor Liao's research related to biological computing by leveraging my skills in process automation, data visualization of scientific data, and educational development (currently developing an intro to Python class with biology examples).

Originally from Brazil, I've called the NYC metropolitan area home since 2018 and feel deeply connected to the city's energy.
